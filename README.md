# SpringBoot-Project-Mount


## 💻 프로젝트 소개
등산을 좋아하는 사람의 입장에서 산의 정보를 검색하고, 등산의 경험을 기록할 수 있는 웹 사이트를 만들어보고자 개발했습니다.

<br>

## ⏱️ 개발 기간
2023.11.13 ~ 2024.02.28

## ⚙️ 개발 환경
- Java
- JDK 17
- IDE: STS 4 
- Framework: Springboot(2.x)
- Database: MySQL
- ORM: MyBatis, JPA
- 형상관리 툴: GitHub
